## Goal
Build a classification model to predict Iris species using sepal and petal measurements.

## Files
- `iris_classification.py` — Full ML pipeline
- `iris.csv` — Dataset

## Models Used
- k-NN
- Logistic Regression
- Decision Tree

## How to Run
1. Install libraries: `pip install numpy pandas matplotlib seaborn scikit-learn joblib`
2. Place `iris.csv` in the same folder
3. Run: `python iris_classification.py`
